# Services
* Randomizer: 
    * *example*: randomize?*time*=**3**&*wordsPerMin*=**170**&*charsPerWord*=**7**
    * *time* -> meditation's duration in minutes
    * *wordsPerMin* -> average number of words per minute
    * *charsPerWord* -> average number of characters in words
